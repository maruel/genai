DashScope exposes an OpenAI-compatible endpoint at:

- International: `https://dashscope-intl.aliyuncs.com/compatible-mode/v1`
- US: `https://dashscope-us.aliyuncs.com/compatible-mode/v1`

No official Go SDK. git clone the official Python SDK from https://github.com/dashscope/dashscope-sdk-python to learn more.

API key env var: `DASHSCOPE_API_KEY` (keys are region-specific).
