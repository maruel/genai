## Scoreboard Recording

To save costs, **only record the latest model of each class**:
- Haiku: only the latest `claude-haiku-*`
- Sonnet: only the latest `claude-sonnet-*`
- Opus: only the latest `claude-opus-*`

Older models (e.g., opus-4-5, opus-4-1, sonnet-4) should retain their existing cassettes
but should **not** be re-recorded unless strictly necessary.
